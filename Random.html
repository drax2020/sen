HELP
  • [-email-] : Reciver Email
  • [-time-] : Date and Time (03/20/2022 12:35:35 am)
  • [-date-] : Date and Time (03/20/2022)
  • [-hour-] : Date and Time (12:35:35 am)
  • [-emailuser-] : Email User (user@emaildomain)
  • [-emaildomain-] : Email Domain (user@mailDOMAIN)
  • [-emaildomainID-] : Email Domain ID (user@mailDomainID.com)
  • [-emailLtd-] : Get ltd from Email User (uk,fr,de)
  • [-randommd5-] : Random MD5
  • [-randomstring-] : Random string (0-9,a-z)
  • [-randomnumber-] : Random number (0-9)
  • [-randomletters-] : Random Letters(a-z)
  • [-randomcountry-] : Random Country name
  • [-randomlink-] : Random Links of textarea

Example
  • Reciver Email = user@domain.com
  • hello [-emailuser-] -> hello User
  • Supported by [-emaildomain-] -> Supported by Domain.com
  • Supported by [-emaildomainID-] -> Supported by Domain
  • your code is [-randommd5-] -> your code is e10adc3949e56e057f20f883e

Other
  • [-AZ-{10}-] : Range string uppercase (A-Z)
  • [-az-{10}-] : Range string tiny (a-z)
